# proxy_server_nsu
My proxy server

#СБОРКА
```
   mkdir build
    cd build
    cmake ..
    make
```  

#НАСТРОЙКА FIREFOX
Настройки -> Основные -> Настройки сети -> Настроить -> Ручная настройка прокси -> HTTP прокси -> 127.0.0.1, порт прокси -> ОК;

(По умолчанию порт стоит 8080)
